Methods in console function

1.console.log()-Mainly used to print the output to the console
examples:
console.log('abc');
console.log(1);

2.console.error()-Used to log error message to the console
examples:
console.error('an error occured');

3.console.warn()-Used to log warning message to the console
examples:
console.warn('This is a warning.');

4.console.table()-This method allows us to generate a table inside a console.
examples:
console.table({'sri':1, 'pk':2});

5.console.count()-This method is used to count the number that the function hit by this counting method.
examples:
console.count(i);

6.console.time() :- Starts a timer (can track how long an operation takes. console.timeEnd() :- Stops the times that was previously started by console.time().
examples
console.time();
console.timeEnd();

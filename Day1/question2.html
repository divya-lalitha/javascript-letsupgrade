var in javascript:

1.var declarations are globally scoped or function/locally scoped.
2.var is function scoped when it is declared within a function. This means that it is available and can be accessed only within that function.
3.var variables can be updated and redeclared
examples:
var name="lalitha";
console.log(name);//prints lalitha
name="divya";
console.log(name);//prints divya

Let in javascript:

1.let can be updated but not re-declared.

let Hello = "say Hi";

Const in javascript

1.Variables declared with the const maintain constant values.
2.const declarations are block scoped.
3.const cannot be updated or re-declared.
examples:

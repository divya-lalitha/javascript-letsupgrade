-------DATA TYPES IN JAVASCRIPT---------

JavaScript has seven datatypes
1.Number
2.String
3.Boolean
4.Undefined
5.Null
6.Object
7.Arrays
Among them primitive data types are: String,Number,Boolean,Undefined,Null
The Non primitive data types are: Object.
1.Number:
JavaScript has only one type of numbers.Numbers can be written with, or without decimals
Example:
var x1 = 67.00; // Written with decimals
var x2 = 45; // Written without decimals
2.String:
Strings are written with quotes. You can use single or double quotes:
Examples:
var st="sunshine";//using double quotes
var st='sunshine';//using single quotes
3.Boolean:
Booleans can only have two values: true or false
Example:
var x = 5;
var y = 5;
var z = 6;
(x == y) // Returns true
(x == z) // Returns false
4.Undefined:
In JavaScript, a variable without a value, has the value undefined. The type is also undefined.
Example:
var vehicle; // Value is undefined, type is undefined
5.Null:
In JavaScript null is "nothing". It is supposed to be something that doesn't exist.
Example:
var city=null; //when to try to print city,it displays null
6.Objects:
A JavaScript object is a collection of named values.Objects are variables too. But objects can contain many values.
Example:
var person = { firstName: "lalitha", lastName: "kotta", };
7.Arrays:
JavaScript arrays are used to store multiple values in a single variable.
Example:
var cars = ["Saab", "Volvo", "BMW"];

METHODS IN STRING:
1.search()-This method returns the position of the first occurrence of a specified text in a string:.
Example:

            var str = "Where there is no pain,there is no gain.";
            var pos = str.search("pain");
            console.log("position is: ",pos);//prints the position of pain

2.substring()-This method extracts the characters from a string, between two specified indices, and returns the new sub string.
Example:

            var str = "Hello Hyderabad!";
            var res = str.substring(1,3);
            console.log("The substring is: ", res);//returns the substring between those indices.

3.replace()-1.This method replaces one value with another value in a string.
2.This method replaces only the first match.
3.This method is case sensitive.To replace case insensitive, use a regular expression with an /i flag.
Example:

           st = "Please visit Bangladesh!";
           var n = str.replace("Bangladesh", "India");
           console.log("After replace string is:",n);//replaces old value with given new value.

4.replaceall()-This method replaces all the occurences of the given string.
Example:

           st1= "hello hobby";
           var s1= st1.replace (/b/g,".");
           console.log("After replacement string is: ",s1);
           // 'hello ho..y'

5.trim()-This removes leading and trailing whitespaces from the string.
Example:

            var s1= "         javascript trim    "; 
            var s2=s1.trim();
            console.log("After trimming :",s2);  //trims all leading spaces

6.slice()-This method returns the parts of string from given beginIndex to endIndex.
Example:

            var t1="abcdefgh";  
            var t2=t1.slice(2,5);
            console.log("After slicing:",t2);//slices from 2 to 5.

METHODS IN ARRAY:
1.every()-checks whether all the given elements in an array are satisfying the provided condition
Example:
        var marks=[50,40,45,37,20]; 
        function check(value)  
        {  
          return value>30;  //return false, as marks[4]=20  
        }  
        console.log((marks.every(check));

2.splice()-This method is used to add/remove the elements to/from the existing array.
Example:

        var arr=["Monday","Tuesday","Saturday","Sunday","Thursday","Friday"];  
        var result=arr.splice(2);  
        console.log("Updated array:"+arr);  
        console.log("Removed element: "+result);

3.sort()-This method is used to arrange the array elements in some order. By default, sort() method follows the ascending order.
Example:

            var arr=["AngularJS","Node.js","JQuery","Bootstrap"]  
            var result=arr.sort();  
            console.log(result);

4.toString()-This method is used for converting and representing an array into string form().
Example:

            var season=["Spring","Autumn","Summer","Winter"];  
            var str=season.toString(); //toString() method implementation  
            console.log("After converting into string:  "+str);

5.from()-creates a new array that holds the shallow copy from an array or iterable object.
Example:

       console.log(Array.from([10, 20, 30]);//output is 10,20,30
